A live menu is a menu that triggers when the user changes the selection. In other words, it loads the selected site immediately. Let's take a look at the difference between a static menu and a live one:

This is a static, non-live menu. The user must press the "Load" button in order to load the selected site. Changing the current selection doesn't trigger the form.

This is a live menu that loads the selected site when the user changes the current selection. Go ahead and change the selection. Also note that a live menu does not need a button (it's optional).
